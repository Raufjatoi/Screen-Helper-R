# 🟨 Screen Helper R

### Snipping, Blurring, and Screen Recording made easy. Simple, fast, and casual! 🚀

Screen Helper R is a Chrome extension designed to help you capture, manage, and record your screen activities with ease. Whether you need to snip a specific element, blur sensitive info, or record a quick demo, we've got you covered.

---

## ✨ Features

*   **🔍 Precision Snipping**: Hover and select any element on the page to capture it with pixel perfection.
*   **🌫️ Privacy Blur**: One click to instantly blur any element, keeping sensitive data private in your screenshots or recordings.
*   **🎥 Screen Recording**: Record your screen with optional camera overlay and audio – perfect for tutorials and bug reports.

## 🚀 How to Install

1.  Download or clone this repo.
2.  Open Chrome and go to `chrome://extensions/`.
3.  Enable **Developer mode** (top right).
4.  Click **Load unpacked** and select the project folder.
5.  Start enhancing your screen experience! 🚀
